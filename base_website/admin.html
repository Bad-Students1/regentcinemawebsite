1Connect to the movie database

This tool looks up real movie titles, posters and ratings for you automatically using TMDB (The Movie Database) — a free service made for exactly this. You only have to do this once.

No API key saved yet
I don't have a key yet — how do I get one?
  1. Go to themoviedb.org/signup and create a free account.
  2. Once logged in, go to your account Settings → API (or visit themoviedb.org/settings/api).
  3. Click "Request an API Key", choose "Developer", and fill in the short form (you can put "personal cinema website" as the use case).
  4. Copy the key labelled "API Key (v3 auth)" — it's a long string of letters and numbers — and paste it above.

It's free and only takes a couple of minutes. This key stays saved in this browser only — it's never sent anywhere except to TMDB.
